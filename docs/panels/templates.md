# Templates & Sharing

Don't want to build a panel from scratch? TicketForge supports a powerful templating system that allows you to save, share, and import configurations instantly.

## Creating a Template
You can turn any existing panel into a template.

1.  Go to the **Panel Editor** for the panel you want to share.
2.  Click the **Template** button in the top action bar.
3.  **Configure Details:**
    *   **Title:** A clear name for the setup (e.g., "Roleplay Application").
    *   **Description:** Explain what the panel does.
    *   **Tags:** Add tags like `Support`, `Gaming`, or `Application` to help others find it.
4.  **Create:** Once created, the template is saved to the community library.

!!! warning "Sensitive Data"
    Templates **do not** save server-specific IDs like Channels or Roles. When you import a template, you will need to re-map these connections.

## Importing a Template
When creating a new panel, you can choose **Community Templates** or **Official Templates**.

1.  Click **Create New Panel**.
2.  Browse the **Desired (Official/Community)** tab.
3.  Click **Preview** on a card to see the flow.
4.  Click **Use Template**.
5.  The **Setup Wizard** will guide you through mapping the required roles and categories.

## Share Codes
If you want to share a configuration privately without publishing it to the community list:

1.  Open the Panel Editor.
2.  Click the **Code** button (The Code Icon).
3.  Copy the generated **Share Code** (Format: `XXXX-XXXX-XXXX`).
4.  Give this code to another user.
5.  They can paste it into the **Import Code** tab when creating a panel.

!!! info "Expiration"
    Share codes expire after **72 hours** for security reasons.
